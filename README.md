# Customer Segmentation Using K-Means Clustering

## Objective
To group customers based on their annual income and spending score using K-Means Clustering.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Dataset
Mall Customers Dataset

## Method
1. Load dataset
2. Select Annual Income and Spending Score
3. Apply K-Means Clustering
4. Visualize customer groups

## Result
Customers were divided into 5 distinct segments based on income and spending behavior.
